#  HR & Employee SQL Analysis

## Project Overview
A complete SQL data analysis project using MySQL Workbench. Designed a relational HR database with 4 linked tables and wrote 15 SQL queries to extract key business insights about employees, salaries, performance and attendance.

## Tools Used
- MySQL Workbench 8.0
- SQL (SELECT, JOIN, GROUP BY, COUNT, AVG, SUM, LIMIT)

## Database Structure
- departments — 6 departments
- employees — 20 employee records
- performance — 2024 performance ratings
- attendance — 2024 attendance records

## Key Results
- Total Employees: 20 (19 Active, 1 Resigned)
- Gender Split: 50% Male, 50% Female
- Top Performer: Efua Appiah (IT) — Score 97/100
- Best Department: Finance — Avg Score 88.0
- Gender Pay Gap: Male avg $7,760 vs Female avg $6,640

## Key Insights
- 80% of staff rated Excellent or Good
- IT department has highest attendance rate (96.9%)
- Sales department needs performance improvement (avg score 73.3)
- Peak hiring years were 2020 and 2021
